# Drop Lists

RuneLite plugin for managing toggleable item lists that swap inventory left-click to **Drop**.

## Usage

1. Open the **Drop Lists** side panel.
2. Create a list and select it.
3. Add items by typing the exact item name and pressing Enter/Add, or by right-clicking inventory items and choosing **Add to drop list**.
4. Toggle a list on to make left-click Drop for its items.

Lists and toggle states persist between client restarts.
